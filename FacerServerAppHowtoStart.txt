How To Start
--==========================================================================================================================

1) Customize pom.xml
2) Customize hibernate.cfg.xml
3) Customize web.xml
4) Choose table/tables for your task
5) Create entity(getters/setters) clas/clasess - facerserver.dao package
6) Design get,post,put,delete methods for your entity clasess
7) Create DAO clas/clasess - facerserver.dao package
8) Modify FacerResource.java clas facerserver package 
	add URI <mywebservice> to FacerResource.java - this action will publish your method  from DAO class as Web Service
10)Run as - maven build install
11)Run as - Run on the server
12)Go url : http://localhost:8080/facerServer/webapi/<mywebservice>/
